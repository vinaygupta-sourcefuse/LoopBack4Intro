# LoopBack4Intro


for installing loopback4

```sudo npm install -g @loopback/cli```

after this 

```lb4 app my-loopback-app```  only loweracse required 

```cd my-loopback-app```

```npm start```

now

    lb4 model
    lb4 datasource
    lb4 repository
    lb4 controller
